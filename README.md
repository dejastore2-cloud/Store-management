## ⚡ تشغيل محلي فوري

```bash
cd webapp
pip install -r requirements.txt
python app.py
```
ثم افتح المتصفح على: **http://localhost:5000**

**بيانات الدخول الافتراضية:**
- المستخدم: `admin`
- كلمة المرور: `admin1234`

> ⚠️ **غيّر كلمة المرور فوراً** من الإعدادات → إدارة المستخدمين بعد أول دخول.

## 🌐 النشر كـ Web App برابط دائم (مجاناً)

### الطريقة الأولى: Render.com (الأسهل)

1. ارفع المجلد `webapp` على GitHub (repo جديد)
2. اذهب إلى [render.com](https://render.com) وأنشئ حساباً مجاناً
3. اضغط **New → Web Service** واختر الـ repo
4. الإعدادات:
   - **Runtime:** Python 3
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn app:app`
5. اضغط **Deploy** — ستحصل على رابط مثل: `https://gift-shop.onrender.com`

### الطريقة الثانية: PythonAnywhere (بديل مجاني)

1. أنشئ حساباً على [pythonanywhere.com](https://pythonanywhere.com)
2. ارفع ملفات المشروع عبر Files أو git clone
3. من **Web** → أضف تطبيق Flask جديد وأشر لـ `app.py`

## 📱 تثبيت التطبيق على الهاتف (PWA)

بعد الحصول على الرابط:
1. افتح الرابط في **Chrome** على الموبايل
2. اضغط على قائمة المتصفح (⋮)
3. اختر **"إضافة إلى الشاشة الرئيسية"**

سيظهر التطبيق كأيقونة مستقلة على الهاتف ويعمل بدون شريط المتصفح.

## ✨ المميزات الكاملة

| الميزة | التفاصيل |
|--------|----------|
| 🏪 الأصناف | إضافة/تعديل/حذف، صور، باركود، أقسام، استيراد Excel |
| 🛒 المبيعات | فاتورة لحظية، بحث بالاسم/الباركود، خصم، طباعة، إجمالي الربح |
| 📦 المشتريات | فواتير الشراء، تحديث مخزون تلقائي، إدارة الموردين |
| 💸 المصاريف | تسجيل المصاريف، أنواع مخصصة، تصدير |
| 📊 التقارير | مبيعات/مشتريات/مصاريف/أرباح، تصدير Excel وPDF |
| 📋 الجرد | مقارنة كميات النظام بالفعلية، تطبيق وتحديث |
| 💰 الخزنة | تتبع كامل للوارد والمنصرف من كل مصدر |
| 👥 المستخدمون | نظام صلاحيات مدير/موظف، سجل نشاط |
| 🎨 التخصيص | ألوان، شعار، وضع ليلي، تصميم شاشة الدخول (4 قوالب) |
| 💾 النسخ الاحتياطي | تنزيل .db كامل أو Excel شامل، استعادة بنقرة |
| 📥 استيراد Excel | أصناف ومبيعات مع معاينة قبل الحفظ |
| 📱 PWA | يعمل على الموبايل والكمبيوتر، قابل للتثبيت |

## 🔧 المتطلبات التقنية

- Python 3.10+
- Flask 3+
- SQLite (مدمج مع Python، لا يحتاج تثبيت)
- المكتبات: openpyxl, reportlab, Pillow, gunicorn
